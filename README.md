# 📱 Cassenje App - O pilar da educação 

A **Cassenje** é a plataforma definitiva para o estudante angolano. Este repositório contém os recursos e informações sobre a nossa aplicação móvel, focada em democratizar o acesso a livros educativos e conteúdos premium.

## ✨ Funcionalidades Principais
* **Leitura Offline:** Baixe os seus livros e estude em qualquer lugar, sem gastar dados.
* **Conteúdo Premium:** Acesso a resumos, dicas e questões por apenas 250 Kz.
* **Assinaturas Flexíveis:** Planos diários, quinzenais e mensais para acesso ilimitado à biblioteca.
* **Banco de Provas:** Estude com exames passados das maiores universidades de Angola (UAN, UTANGA, UnIA).

## 🚀 Como baixar e usar
1. Vá para a seção de [Releases](link-da-release) deste repositório.
2. Baixe o arquivo `.apk` mais recente.
3. Instale no seu dispositivo Android.
